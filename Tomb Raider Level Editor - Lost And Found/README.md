# Tomb Raider Level Editor - Lost And Found AutoSplitter Component

LiveSplit AutoSplitter Component for Tomb Raider Level Editor - Lost And Found v4.

## Contents

- `ASL/` contains the original ASL script.
- `Source/` contains the LiveSplit Component source project.
- `Components/` contains the compiled LiveSplit Component DLL.
- `CheatEngine/` is reserved for the Cheat Engine table.

## Installation

Copy `Components/TRLELostAndFoundv4.dll` into your LiveSplit `Components` folder, then restart LiveSplit.
